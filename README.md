# bpsdaweb
Evaluation of metagenomic binning

Dependencies (python 3):
* flask
* flask-sqlalchemy
* flask-restful

Instructies:  
1. Maak database aan met `python3 create_db.py`  
2. Start applicatie met `python3 app.py`  
3. Open webbrowser en ga naar 127.0.0.1:5000
